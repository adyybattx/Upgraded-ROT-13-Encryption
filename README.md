## Upgraded ROT-13 Encryption

<div align="left">

$\color{goldenrod}{\text{A custom upgrade from the vanilla ROT-13.}}$
 
</div>

---

An attempt to upgrade the $$\color{pink}{\text{ROT-13 Encryption}}$$, which pushes the $$\color{blue}{\text{full ASCII range}}$$ by `+13`<br> and encodes output as $$\color{skyblue}{\text{zero-padded 3-digit integers}}$$.

Classical ROT-13 only shifts $$\color{darkorange}{\text{A–Z}}$$. This shifts the $$\color{red}{\text{entire ASCII space}}$$.

The numerical encoding layer on top means even the output format $$\color{green}{\text{carries no recognizable structure.}}$$

---

Its a small but useful update from the vanilla ROT-13.

**Happy Encrypting.**
